# Google Maps Scraper - Google Maps Data Extraction Tool

A powerful Chrome extension for extracting business data, reviews, and contact information from Google Maps.

## Features
- Extract business information (name, address, phone, website)
- Scrape customer reviews and ratings
- Bulk extract phone numbers from search results
- Export data to CSV format
- Support for latest Google Maps UI

## Technologies Used
- Chrome Extension Manifest V3
- JavaScript ES6+
- HTML5/CSS3
- Chrome APIs
